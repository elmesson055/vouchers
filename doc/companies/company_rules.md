# Regras para Empresas

## 1. Cadastro

### Obrigatório
- Nome
- CNPJ
- Endereço completo
- Responsável

### Opcional
- Logo
- Cores personalizadas
- Observações

## 2. Configurações

### Limites
- Número máximo de usuários
- Vouchers extras por mês
- Vouchers descartáveis por mês

### Personalizações
- Horários específicos
- Tipos de refeição
- Valores diferenciados
